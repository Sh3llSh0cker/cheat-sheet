***Packages + Dependencies***

```sh
1) sudo apt install php-cli build-essential -y **will need these**
2) git clone https://github.com/phoronix-test-suite/phoronix-test-suite.git /opt/phoronix-test-suite **or where ever you want**
3) cd /opt/phoronix-test-suite
4) sudo apt install php-xml php-gd php-bz2 php-curl -y **PHP extensions are needed**
5) ./phoronix-test-suite version **Phoronix Test Suite v10.8.5**
6) ./phoronix-test-suite **run Suite**
```
